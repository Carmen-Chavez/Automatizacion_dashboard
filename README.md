# Automatización con Tableau

## Objetivo Principal
Crear un dashboard interactivo que automatice el análisis de datos y proporcione insights visuales para la toma de decisiones.

## Proceso de Trabajo

- Preparación de datos localmente
- Procesar y limpiar los datos
- Creación del dashboard
- Diseñar visualizaciones en Tableau Desktop
- Publicación en Tableau Public
- Hacer dashboard accesible online

## Habilidades a desarrollarar
- Automatización de procesos de análisis de datos
- Visualización avanzada con Tableau
- Storytelling con datos a través de dashboards
- Publicación y compartición de análisis
- Documentación técnica de proyectos
